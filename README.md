# Auth-Roles API Microservices

1. Clone project from github
2. ```yarn install```
3. Clone file ```.env.template``` and rename it to ```.env```
4. use your environment variables ( for test reasons they will stay in this file)
5. up docker
```
docker-compose up -d
```
6. run : ```yarn start:dev```
